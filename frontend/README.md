# Oliver App

Estrutura unificada com frontend e backend separados.

## Frontend

- Pasta: `frontend/`
- `.env` esperado:
```
VITE_API_URL=http://localhost:8081
```
- Rodar local:
```
npm i
npm run dev
```

## Backend

- Pasta: `backend/`
- `.env` esperado:
```
MONGODB_URI=
PORT=8081
JWT_SECRET=
CORS_ORIGIN=http://localhost:8080
```
- Rodar local:
```
npm i
npm run dev
```
